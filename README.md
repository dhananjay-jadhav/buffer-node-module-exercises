# buffer-node-module-exercises

This repository contains exercises and code samples for exploring the Buffer module in Node.js.

## Features

- Examples of using Node.js Buffers
- Explanatory comments and sample code
- Exercises for hands-on learning

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/dhananjay-jadhav/buffer-node-module-exercises.git
   ```
2. Install dependencies (if any):
   ```bash
   npm install
   ```
3. Run the sample code:
   ```bash
   node example.js
   ```

## Folder Structure

- `examples/` — Common Buffer operations, usage patterns
- `exercises/` — Problems and solutions for practice

## Exercises

Below are practice exercises for the Buffer Module (adapted from [here](https://github.com/dhananjay-jadhav/Node-Learning/blob/main/Buffer%20Module/Small%20Practice%20Exercises%20for%20Buffer%20Module.md)):

### Buffer Module Practice Exercises

1. **Create a buffer from a string and print its contents.**
2. **Write a string to a buffer and convert it to JSON.**
3. **Concatenate two buffers.**
4. **Compare two buffers to determine if they are equal.**
5. **Copy data from one buffer to another.**
6. **Slice a buffer and print the result.**
7. **Write data to a buffer with encoding.**
8. **Reading data from user input and storing it in a buffer.**
9. **Check the length of a buffer.**
10. **Fill a buffer with a sequence of values.**

*For implementation guidance, refer to the [Node.js Buffer documentation](https://nodejs.org/api/buffer.html).*
